# `tinye`

a tiny code editor written in Rust

## features

- basic character insertion/deletion
- cursor movement
- file saving
- line numbers
- indentation inheritance
- no copy/paste
- no multiple buffers
- no scrolling window
- will probably error randomly

## should i use it? (please don't)

yes if: you want to try coding barebones-ly (or just use vi for that tbh)

(`tinye` is really inefficient so please don't use it as a reference)
