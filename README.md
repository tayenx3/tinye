# `tinye`

a tiny code editor written in Rust

## installation

```bash
cargo install --git https://github.com/tayenx3/tinye.git
```

(you can install cargo at [rustup.rs](https://rustup.rs))

## should i use it? (please don't)

yes if: you want to try coding barebones-ly (or just use vi for that tbh)
